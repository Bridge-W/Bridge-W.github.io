## 为什么选择 Astro？

Astro 是一个**内容驱动的网站构建框架**，它默认发送零 JavaScript，只发送必要的 HTML 和 CSS。

### 核心优势

1. **性能卓越** - 默认零 JS，Core Web Vitals 得分极高
2. **内容优先** - 原生支持 Markdown 和 MDX
3. **框架无关** - 可集成 React、Vue、Svelte 等组件
4. **群岛架构** - 只在需要的地方添加交互性
